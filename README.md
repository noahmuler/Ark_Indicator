# Ark Indicator - Advanced Smart Money Concepts

## Project Overview

The Ark Indicator is a comprehensive TradingView Pine Script tool that implements advanced Smart Money Concepts (SMC) trading strategies. Designed for professional traders, this indicator combines institutional-grade analysis tools with an intuitive interface for precise market structure analysis.

## 🚀 Key Features

### 📊 Market Structure Analysis
- **Market Structure Shifts (CHoCH/BOS)**: Automatic detection of trend changes and breakouts
- **Order Blocks (OB)**: Institutional order block identification with bullish/bearish classification
- **Breaker Blocks**: Advanced breaker block detection for reversal trading
- **Fair Value Gaps (FVG)**: Smart FVG detection with automatic filtering and mitigation tracking

### 🕐 Session Management
- **Global Sessions**: Sydney, Asia, London, NY AM, NY PM with automatic daylight savings
- **UTC-4 NY Time Zone**: Perfectly aligned with US trading hours

## 🛠️ Installation & Setup

### Requirements:
- TradingView Pro or Premium account
- Pine Script v6 compatibility
- UTC-4 timezone setting recommended

### Installation Steps:
1. Copy the complete Pine Script code
2. Open TradingView Pine Editor
3. Paste and save the script
4. Add to your preferred chart
5. Configure settings in the inputs panel

### Initial Configuration:
1. Set your timezone to UTC-4 (NY Time)
2. Enable desired sessions in Session Settings
3. Configure Order Block and FVG parameters
4. Adjust colors to match your preference
5. Set up alert conditions if needed

## 📚 User Guide

### Session Settings:
- **Sydney**: 5:00 PM - 2:00 AM NY Time
- **Asia**: 7:00 PM - 4:00 AM NY Time  
- **London**: 3:00 AM - 7:00 AM NY Time
- **NY AM**: 8:00 AM - 12:00 PM NY Time
- **NY PM**: 12:00 PM - 4:00 PM NY Time

### Order Block Configuration:
- **Swing Detection Length**: Adjust sensitivity (default: 5)
- **Mitigation Price**: Choose between close or wick
- **Display Options**: All, First, or Last blocks
- **Color Customization**: Separate colors for bullish/bearish blocks

### Fair Value Gap Settings:
- **Width Filter**: ATR multiplier for FVG filtering (default: 1.2)
- **Remove Mitigated**: Auto-delete filled FVGs
- **Extension**: Extend FVGs to current bar
- **Display Control**: Show All, First, or Last FVGs

### Market Structure Shifts:
- **Detection Length**: Pivot sensitivity (default: 7)
- **Display Options**: Control CHoCH/BOS visibility
- **Text Labels**: Enable/disable structure text
- **Color Schemes**: Customizable bullish/bearish colors
